# 3D model viewer

3D model viewer based on three.js

## modelviewer

| name       | type    | default | required |
|------------|---------|---------|----------|
| aoto       | boolean | false   | -        |
| background | string  | -       | -        |
| type       | string  | -       | Y        |
| model      | link    | -       | Y        |
